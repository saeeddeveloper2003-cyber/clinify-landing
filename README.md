# 🏥 Clinify

A modern landing page for a medical and beauty platform, built to practice clean frontend architecture, UI implementation, and modern web development workflows.

> 🚧 **Project Status:** Under Active Development

## 🌐 Live Demo

🔗 https://clinfay-landing.vercel.app/

---

## 📖 About

Clinify is a frontend project focused on creating a clean and modern user interface for a medical and beauty platform.

The current version is a **desktop-first landing page** developed using **HTML, CSS, and JavaScript**.

This project is continuously improving and will eventually evolve into a complete full-stack application.

---

## ✨ Current Features

- Modern Medical UI
- RTL (Right-to-Left) Layout
- Clean HTML Structure
- Organized CSS Architecture
- Vanilla JavaScript
- Custom Typography (Peyda Font)
- Reusable Components
- Git & GitHub Workflow
- Vercel Deployment
